# Save Our Shores

A climate awareness website highlighting threats to Goa's coastline — built during the Lenovo LEAP frontend development internship.

## Pages
- **Home** – Hero section introducing the coastline crisis
- **Threats** – Sea-level rise, coastal erosion, and saltwater intrusion overview
- **Coastline Map** – Interactive map visualizing sea-level rise areas along Goa's coast
- **Solutions** – Mitigation efforts including mangrove restoration and sustainable coastal management
- **Act Now** – Ways to support local conservation efforts

## Tech Stack
- HTML
- CSS
- JavaScript
